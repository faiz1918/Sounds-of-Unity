About This Project

Sounds of Unity was created by Muhammed Faiz as a final project for MMP100, showcasing how music, art, and storytelling can unite communities.

The project combines:

  • Audio — Edited recordings of music, voices, and sounds from the community.
  • Images — Photographs and illustrations representing community life.
  • Animation & Web Design — Smooth CSS effects and clean layout to create an engaging user experience.

Credits and Media Licenses
